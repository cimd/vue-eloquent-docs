An interface for Vue 3 an Laravel Eloquent models through API requests.

## Installation

### Vue 3
`yarn add @konnec/vue-eloquent`

#### Vue Requirements
1. Vue 3

### Laravel
`composer require konnec/vue-eloquent-api`

#### Laravel Requirements
1. Laravel 10
2. PHP ^8.1
